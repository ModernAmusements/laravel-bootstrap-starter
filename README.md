# Laravel Mix Starter Kit - Shady Tawfik

## What's inside?

It's a pretty simple project, it's a starter kit based on [Laravel Mix](https://github.com/JeffreyWay/laravel-mix)

Include: 

- Vue
- Bootstrap 4
- Axios
- Popper.js
- lodash
- jQuery
- cross-env

## Installation

1. `npm install`
2. `npm run dev`
3. Add `mix.browserSync()` to `webpack.mix.js`
